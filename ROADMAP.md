# C/C++ 代码行统计工具 - 项目路线图

## 项目概述

cc_loc_tool 是一个使用 Rust 开发的 C/C++ 代码行统计工具，提供 GUI 界面，用于快速扫描、统计和分析 C/C++ 项目的代码行、注释行和空白行。

## 开发优先级

- 🟢 **高优先级**：核心功能增强和用户体验改进
- 🟡 **中优先级**：性能优化和扩展性
- 🔴 **低优先级**：非核心功能和高级特性
