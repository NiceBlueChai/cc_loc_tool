# C/C++ 代码行统计工具 - 项目路线图（可执行版）

## 项目概述

`cc_loc_tool` 是一个使用 Rust 开发的代码统计与质量分析工具，提供 GUI 与 CLI，用于扫描项目并输出代码行、注释行、空白行及复杂度指标。

## 当前状态

### 已完成能力

- 多语言支持、配置持久化、导出（CSV/JSON/HTML）、CLI、进度显示
- 深色主题、响应式布局、统计图表、文件预览
- 并行扫描、内存优化、扫描缓存
- 复杂度分析、文件类型自定义
- 单元测试覆盖、CI/CD 基线
